**Goal:** Rework the React app from the legacy XSight trading-copilot UI into the **X Cup
prediction market** — 8 screens (DESIGN §5/§6), World Cup visual theme, real team
flags/logos, OKX Wallet connect, and the approve → stake → claim flow.

**Stack (existing):** React 18 + Vite + Tailwind 4 + zustand + motion + recharts +
lucide. Reuse `components/common/*` primitives and `components/cup/TeamLogo.tsx`
(already renders real flags via `lib/teamMeta`).

**Backend (ready, Plans 1–3):** `GET /api/markets`, `/api/markets/:id`,
`/api/markets/:id/position`, `/api/markets/indexer`, `POST /api/markets/:id/stake-tx`,
`GET /api/markets/:id/claim-tx`; `/api/cup/*` (fixtures, ai-edge, resolver, fan-score,
fanpass, track-proof, settlement-log, contract). Vite proxies `/api` → `:8787`.
